# Antimicrobial Analyzer

**Author:** Ali Dahou

A desktop Python application to record and analyze antifungal activity of halophilic bacteria.  
Features:
- Record inhibition zone (mm), control (KR), and concentration (CFU/ml)
- Compute Percentage Growth Inhibition (PGI%)
- Compare isolates vs fungi with grouped bar charts, boxplots and scatter plots
- Generate PDF reports (plots + PGI summary)
- Import/export CSV datasets, image previews
